Static: It is a default position, even if we do not specify any position or value for elements, the system automatically sets the default values, and they are unchangeable. The system follows first come first out strategy. That means whatever comes first that will be displayed first. Syntax: position:static Relative: If you want to change the position of the elements, you can use relative. When you are changing elements' positions to a relative, it changes the position of the elements by using the below attributes. Attributes are as follows Alignment: In this attribute, you can give the values like top, bottom, right or left according to that element position changes. Size: Mention in the pixels.
